# 🔐 Password Manager

A secure and simple **Password Manager Web Application** that allows users to store and manage their website credentials safely. The application enables users to save, view, edit, and delete login credentials for different websites.

This project demonstrates full-stack development using **React, Node.js, Express, and MongoDB**.

---

## 🚀 Features

- ➕ Add new website credentials  
- 👀 Show / hide passwords  
- ✏️ Edit saved credentials  
- ❌ Delete stored passwords  
- 📋 Copy username or password to clipboard  
- 💾 Persistent storage using MongoDB  
- 🔔 Toast notifications for user actions  
- ⚡ Fast frontend powered by Vite + React
